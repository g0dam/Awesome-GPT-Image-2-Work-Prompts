# Competitive positioning ad matrix

**Target user:** growth marketer

**Use case:** Create a competitive positioning visual for a B2B SaaS landing page or paid campaign

## Prompt

Create a 16:9 business-ready image for a growth marketer. Goal: Create a competitive positioning visual for a B2B SaaS landing page or paid campaign. Deliverable: Competitive positioning ad matrix. Visual direction: A two-axis competitive positioning matrix with four quadrants, a highlighted product zone, competitor-neutral placeholders, proof chips, SaaS dashboard fragments and a CTA strip. Layout and content requirements: show a clear two-axis matrix with concise quadrant labels; use competitor-neutral placeholders rather than real brand names; include one highlighted value zone, three proof chips and a CTA area; keep hierarchy readable for both GitHub preview and website card crops; use realistic SaaS UI fragments without copying any product interface. Use precise layout language, professional work context, crisp readable labels, realistic interface or document details where useful, and enough negative space for website cards and GitHub preview crops. Avoid real company logos, competitor names, celebrity likenesses, generic stock-photo people, unreadable text, inflated performance claims, cluttered axes, distorted UI.
