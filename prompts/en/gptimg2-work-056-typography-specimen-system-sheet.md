# Typography specimen system sheet

**Target user:** design systems lead

**Use case:** Document product UI type scale, numeric examples and accessibility notes

## Prompt

Create a 4:3 business-ready image for a design systems lead. Goal: Document product UI type scale, numeric examples and accessibility notes. Deliverable: Typography specimen system sheet. Visual direction: A design-system documentation spread with type scale, numeric table samples, button labels, chart annotation examples, contrast notes, spacing tokens and baseline grid hints. Layout and content requirements: use placeholder text and neutral labels only; include headline, body, numeric and UI component examples; show accessibility and contrast notes without legal fine print; use a precise documentation grid; keep typography crisp and readable. Use precise layout language, professional work context, crisp readable labels, realistic interface or document details where useful, and enough negative space for website cards and GitHub preview crops. Avoid decorative poster excess, fake logos, tiny unreadable samples, inconsistent baselines, celebrity references, clutter.
