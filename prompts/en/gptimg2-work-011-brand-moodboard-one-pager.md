# Brand moodboard one-pager

**Target user:** brand designer

**Use case:** Create a brand moodboard one-pager for a fintech startup

## Prompt

Create a 结构化情绪板，含 logo 占位、字体块、色彩样本、图像参考和使用说明。 business-ready image for a brand designer. Goal: Create a brand moodboard one-pager for a fintech startup. Deliverable: Brand moodboard one-pager. Visual direction: A structured moodboard with logo placeholder, typography blocks, color chips, imagery references and usage notes. Use precise layout language, keep text short and legible, show realistic work context, include enough negative space for website cropping, avoid generic stock-photo smiles, fantasy elements, celebrity likenesses, copyrighted brands, unreadable text, clutter, warped hands, and exaggerated claims.
