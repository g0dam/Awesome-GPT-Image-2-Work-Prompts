# Event floor plan signage kit

**Target user:** community manager

**Use case:** Plan a professional meetup layout, signage system and staff operations cues

## Prompt

Create a 16:9 business-ready image for a community manager. Goal: Plan a professional meetup layout, signage system and staff operations cues. Deliverable: Event floor plan signage kit. Visual direction: A simplified venue map with registration, main stage, breakout tables, demo booths, networking zone, quiet area, sponsor wall, signage samples, badge system and accessibility notes. Layout and content requirements: use neutral event and venue placeholders only; show wayfinding signage and badge color system; include staff cue cards and accessibility notes; keep floor plan labels legible; make it practical for operations planning. Use precise layout language, professional work context, crisp readable labels, realistic interface or document details where useful, and enough negative space for website cards and GitHub preview crops. Avoid real venue names, brand logos, crowded party scenes, tiny labels, decorative-only event graphics, confusing floor plans.
