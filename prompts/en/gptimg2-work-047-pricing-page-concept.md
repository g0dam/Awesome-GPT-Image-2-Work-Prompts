# Pricing page concept

**Target user:** founder

**Use case:** Create a pricing page concept visual for a three-tier SaaS plan

## Prompt

Create a 网站截图式定价布局，含三档方案、用量说明、信任条和清晰推荐方案。 business-ready image for a founder. Goal: Create a pricing page concept visual for a three-tier SaaS plan. Deliverable: Pricing page concept. Visual direction: Website screenshot-style pricing layout with three plans, usage notes, trust strip and clear recommended plan. Use precise layout language, keep text short and legible, show realistic work context, include enough negative space for website cropping, avoid generic stock-photo smiles, fantasy elements, celebrity likenesses, copyrighted brands, unreadable text, clutter, warped hands, and exaggerated claims.
