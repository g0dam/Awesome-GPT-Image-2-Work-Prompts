# Data handling compliance scenario poster

**Target user:** corporate trainer

**Use case:** Explain data handling rules through realistic workplace scenarios

## Prompt

Create a 4:3 business-ready image for a corporate trainer. Goal: Explain data handling rules through realistic workplace scenarios. Deliverable: Data handling compliance scenario poster. Visual direction: A corporate learning poster with three workplace scenarios, each showing risk cue, approved action, escalation path and do-not-do note. Layout and content requirements: cover customer data export, vendor file sharing and AI tool input review; use accessible scenario cards with approval and warning states; keep language practical and non-alarmist; include a short footer checklist; use no real company names. Use precise layout language, professional work context, crisp readable labels, realistic interface or document details where useful, and enough negative space for website cards and GitHub preview crops. Avoid fear-mongering, dense legal fine print, real logos, unreadable text, celebrity likenesses, cluttered compliance charts.
