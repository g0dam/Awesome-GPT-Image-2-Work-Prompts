# Renewal value recap QBR

**Target user:** customer success manager

**Use case:** Prepare a quarterly business review visual that summarizes adoption, outcomes and renewal confidence

## Prompt

Create a 4:3 business-ready image for a customer success manager. Goal: Prepare a quarterly business review visual that summarizes adoption, outcomes and renewal confidence. Deliverable: Renewal value recap QBR. Visual direction: A polished QBR recap with before-and-after outcomes, adoption metrics, value delivered, open risks, quote placeholders, next-quarter opportunities and renewal confidence score. Layout and content requirements: use fictional customer placeholders only; separate outcomes, adoption, value delivered and risks; include next-quarter opportunities and confidence score; use cautious, evidence-based language; make it executive-readable. Use precise layout language, professional work context, crisp readable labels, realistic interface or document details where useful, and enough negative space for website cards and GitHub preview crops. Avoid guaranteed ROI claims, real logos, stock-photo people, confetti celebration, tiny text, cluttered QBR slides.
