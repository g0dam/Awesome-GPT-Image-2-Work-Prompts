# Logo usage guide spread

**Target user:** brand designer

**Use case:** Create a logo usage guide spread for a professional services firm

## Prompt

Create a 双页编辑版式，展示安全区、最小尺寸、正确和错误用法，以及沉稳企业视觉语言。 business-ready image for a brand designer. Goal: Create a logo usage guide spread for a professional services firm. Deliverable: Logo usage guide spread. Visual direction: Two-page editorial spread showing safe area, minimum size, correct and incorrect usage, and calm corporate visual language. Use precise layout language, keep text short and legible, show realistic work context, include enough negative space for website cropping, avoid generic stock-photo smiles, fantasy elements, celebrity likenesses, copyrighted brands, unreadable text, clutter, warped hands, and exaggerated claims.
