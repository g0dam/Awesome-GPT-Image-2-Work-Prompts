# Sales one-pager cover

**Target user:** sales lead

**Use case:** Create a sales one-pager cover for a cybersecurity service

## Prompt

Create a 高级 B2B 封面，含分层网络图、安全态势评分和干净标题区域。 business-ready image for a sales lead. Goal: Create a sales one-pager cover for a cybersecurity service. Deliverable: Sales one-pager cover. Visual direction: Premium B2B cover with layered network diagram, security posture score and clean headline area. Use precise layout language, keep text short and legible, show realistic work context, include enough negative space for website cropping, avoid generic stock-photo smiles, fantasy elements, celebrity likenesses, copyrighted brands, unreadable text, clutter, warped hands, and exaggerated claims.
