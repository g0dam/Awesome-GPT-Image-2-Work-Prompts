# Warehouse process simulation storyboard

**Target user:** training designer

**Use case:** Teach a new warehouse receiving workflow through scenario panels

## Prompt

Create a 16:9 business-ready image for a training designer. Goal: Teach a new warehouse receiving workflow through scenario panels. Deliverable: Warehouse process simulation storyboard. Visual direction: A six-panel training storyboard from delivery arrival to inspection, scanning, exception routing, inventory update and supervisor sign-off. Layout and content requirements: show six sequential panels with role badges and decision points; include timing markers, exception routing and safety reminders; use realistic warehouse context without unsafe behavior; make arrows and process order clear; keep text short and training-friendly. Use precise layout language, professional work context, crisp readable labels, realistic interface or document details where useful, and enough negative space for website cards and GitHub preview crops. Avoid comic entertainment styling, unsafe warehouse practices, real company logos, excessive text, distorted equipment, cluttered panels.
