**Target user** ecommerce UX designer

**Use case** Design a product detail page blueprint with image, benefits, specs and trust modules

## Prompt

Create a 4:3 business-ready image for a ecommerce UX designer. Goal: Design a product detail page blueprint with image, benefits, specs and trust modules. Deliverable: Product detail page content system. Visual direction: A high-fidelity product detail page blueprint with hero image zone, gallery, benefit stack, spec accordion, comparison strip, trust modules and sticky add-to-cart bar. Use precise visual design language: define focal point, grid, hierarchy, typography, color role, spacing, annotations, crop safety, and realistic work context. Keep labels short and readable; make the artifact useful for a real professional review, campaign, handoff, or operating meeting. Avoid real brand names, copied logos, celebrity likenesses, private data, fake guarantees, unreadable microtext, clutter, distorted UI, generic stock-photo people, and purely decorative entertainment styling.
