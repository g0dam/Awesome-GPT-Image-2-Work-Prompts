# Partner announcement visual

**Target user:** community manager

**Use case:** Create a partner announcement visual for two SaaS companies

## Prompt

Create a 平衡的联合品牌视觉，含双 logo 区、集成示意、共同价值主张和公告标题。 business-ready image for a community manager. Goal: Create a partner announcement visual for two SaaS companies. Deliverable: Partner announcement visual. Visual direction: Balanced co-brand visual with two logo zones, integration diagram, shared value proposition and announcement headline. Use precise layout language, keep text short and legible, show realistic work context, include enough negative space for website cropping, avoid generic stock-photo smiles, fantasy elements, celebrity likenesses, copyrighted brands, unreadable text, clutter, warped hands, and exaggerated claims.
