# Sales objection-handling battlecard

**Target user:** account executive

**Use case:** Create a practical enablement card for handling buyer objections

## Prompt

Create a 4:3 business-ready image for a account executive. Goal: Create a practical enablement card for handling buyer objections. Deliverable: Sales objection-handling battlecard. Visual direction: A compact enablement card with objection categories, response framing, proof asset placeholders, competitor-neutral comparison cues, discovery questions and next-step checklist. Layout and content requirements: cover budget, timing, need and competition objections; separate response framing from proof assets and questions; keep tone consultative rather than manipulative; use short bullets and clear sections; avoid real competitor names. Use precise layout language, professional work context, crisp readable labels, realistic interface or document details where useful, and enough negative space for website cards and GitHub preview crops. Avoid real competitor names, inflated claims, manipulative fear tactics, stock-photo people, tiny text, cluttered battlecard grids.
