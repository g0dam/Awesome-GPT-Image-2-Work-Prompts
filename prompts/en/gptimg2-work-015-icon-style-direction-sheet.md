# Icon style direction sheet

**Target user:** brand designer

**Use case:** Create an icon style direction sheet for a SaaS analytics product

## Prompt

Create a 多行一致图标，含描边规则、填充变体、隐喻说明和真实 UI 表面示例。 business-ready image for a brand designer. Goal: Create an icon style direction sheet for a SaaS analytics product. Deliverable: Icon style direction sheet. Visual direction: Rows of consistent icons with stroke rules, filled variants, metaphor notes and examples placed on real UI surfaces. Use precise layout language, keep text short and legible, show realistic work context, include enough negative space for website cropping, avoid generic stock-photo smiles, fantasy elements, celebrity likenesses, copyrighted brands, unreadable text, clutter, warped hands, and exaggerated claims.
