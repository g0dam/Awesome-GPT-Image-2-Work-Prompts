# Data visualization style guide

**Target user** analytics lead

**Use case** Document approved chart types, color rules and annotation patterns for dashboards

## Prompt

Create a 4:3 business-ready image for a analytics lead. Goal: Document approved chart types, color rules and annotation patterns for dashboards. Deliverable: Data visualization style guide. Visual direction: A chart governance sheet with approved chart types, color rules, axis labels, annotations, chart do/dont cards, spacing tokens and cleanup example. Use precise visual design language: define focal point, grid, hierarchy, typography, color role, spacing, annotations, crop safety, and realistic work context. Keep labels short and readable; make the artifact useful for a real professional review, campaign, handoff, or operating meeting. Avoid real brand names, copied logos, celebrity likenesses, private data, fake guarantees, unreadable microtext, clutter, distorted UI, generic stock-photo people, and purely decorative entertainment styling.
