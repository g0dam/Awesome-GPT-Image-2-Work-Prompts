# Culture values card set

**Target user:** people ops manager

**Use case:** Create a culture values card set for an internal portal

## Prompt

Create a 四张价值观卡片，含职场场景、简短行为示例和一致视觉系统。 business-ready image for a people ops manager. Goal: Create a culture values card set for an internal portal. Deliverable: Culture values card set. Visual direction: Set of four values cards with workplace scenes, short behavior examples and a coherent visual system. Use precise layout language, keep text short and legible, show realistic work context, include enough negative space for website cropping, avoid generic stock-photo smiles, fantasy elements, celebrity likenesses, copyrighted brands, unreadable text, clutter, warped hands, and exaggerated claims.
