# Skills matrix career path visual

**Target user:** people ops manager

**Use case:** Create an internal talent planning visual for role families and growth paths

## Prompt

Create a 4:3 business-ready image for a people ops manager. Goal: Create an internal talent planning visual for role families and growth paths. Deliverable: Skills matrix career path visual. Visual direction: A role-family grid with proficiency levels, skill clusters, learning moves, promotion readiness signals and manager coaching prompts. Layout and content requirements: show role families and proficiency levels without ranking employees; include learning moves and coaching prompts; use abstract inclusive avatars rather than real faces; keep tables readable and calm; make it feel like an internal planning tool. Use precise layout language, professional work context, crisp readable labels, realistic interface or document details where useful, and enough negative space for website cards and GitHub preview crops. Avoid surveillance vibes, real employee data, ranking shame, cartoon entertainment style, tiny unreadable tables, clutter.
