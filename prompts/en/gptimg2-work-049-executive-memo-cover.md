# Executive memo cover

**Target user:** founder

**Use case:** Create an executive memo cover about entering a new market

## Prompt

Create a 严肃编辑式备忘录封面，含市场信号拼贴、风险机会平衡和克制字体。 business-ready image for a founder. Goal: Create an executive memo cover about entering a new market. Deliverable: Executive memo cover. Visual direction: Serious editorial memo cover with market signal collage, risk/opportunity balance and restrained typography. Use precise layout language, keep text short and legible, show realistic work context, include enough negative space for website cropping, avoid generic stock-photo smiles, fantasy elements, celebrity likenesses, copyrighted brands, unreadable text, clutter, warped hands, and exaggerated claims.
