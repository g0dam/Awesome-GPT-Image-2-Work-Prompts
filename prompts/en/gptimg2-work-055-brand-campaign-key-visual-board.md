# Brand campaign key visual board

**Target user:** brand designer

**Use case:** Shape a quarterly product campaign visual system across web, social and email

## Prompt

Create a 16:9 business-ready image for a brand designer. Goal: Shape a quarterly product campaign visual system across web, social and email. Deliverable: Brand campaign key visual board. Visual direction: A campaign system board with hero lockup area, message tiles, palette strip, image direction thumbnails, motion cues and application previews. Layout and content requirements: use placeholder copy only and no real logos; show three message tiles and three application previews; include a palette strip and motion cue arrows; make the board feel like a real brand review artifact; keep layout disciplined and editorial. Use precise layout language, professional work context, crisp readable labels, realistic interface or document details where useful, and enough negative space for website cards and GitHub preview crops. Avoid copied logos, celebrity images, messy moodboard clutter, decorative-only visuals, unreadable type, fake brand marks.
