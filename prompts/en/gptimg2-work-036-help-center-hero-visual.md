# Help center hero visual

**Target user:** customer success manager

**Use case:** Create a help center hero visual for a SaaS knowledge base

## Prompt

Create a 搜索优先的支持头图，含文章卡、产品截图、友好服务暗示，避免通用耳机客服套路。 business-ready image for a customer success manager. Goal: Create a help center hero visual for a SaaS knowledge base. Deliverable: Help center hero visual. Visual direction: Search-first support hero with article cards, product screenshots, friendly service cues and no generic headset cliché. Use precise layout language, keep text short and legible, show realistic work context, include enough negative space for website cropping, avoid generic stock-photo smiles, fantasy elements, celebrity likenesses, copyrighted brands, unreadable text, clutter, warped hands, and exaggerated claims.
