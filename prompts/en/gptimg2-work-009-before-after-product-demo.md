# Before-after product demo

**Target user:** ecommerce operator

**Use case:** Create a before-after demo graphic for a cleaning product without exaggerated claims

## Prompt

Create a 双栏对比，同一表面和光线，改善真实可信，标签精确，并有小使用说明。 business-ready image for a ecommerce operator. Goal: Create a before-after demo graphic for a cleaning product without exaggerated claims. Deliverable: Before-after product demo. Visual direction: Two-panel comparison with consistent lighting, same surface, realistic improvement, precise labels and a small usage note. Use precise layout language, keep text short and legible, show realistic work context, include enough negative space for website cropping, avoid generic stock-photo smiles, fantasy elements, celebrity likenesses, copyrighted brands, unreadable text, clutter, warped hands, and exaggerated claims.
