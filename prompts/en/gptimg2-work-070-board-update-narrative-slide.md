# Board update narrative slide

**Target user:** founder

**Use case:** Create a monthly investor update slide that frames progress, constraints and decisions needed

## Prompt

Create a 16:9 business-ready image for a founder. Goal: Create a monthly investor update slide that frames progress, constraints and decisions needed. Deliverable: Board update narrative slide. Visual direction: A mature board-deck slide with progress since last update, current constraints, decisions needed, metric cards, milestone timeline, hiring/resource ask and risk register. Layout and content requirements: separate progress, constraints and decisions needed; include metrics, timeline, resource ask and risk register; use fictional placeholders and measured language; make board input requested visually explicit; keep text concise and executive-readable. Use precise layout language, professional work context, crisp readable labels, realistic interface or document details where useful, and enough negative space for website cards and GitHub preview crops. Avoid startup hype clichés, real company names, dense paragraphs, dramatic imagery, tiny tables, cluttered board slides.
