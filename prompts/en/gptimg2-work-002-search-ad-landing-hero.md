# Search ad landing hero

**Target user:** growth marketer

**Use case:** Create a high-converting hero image for a search-ad landing page

## Prompt

Create a 干净的对比场景，旧的手工流程淡出，聚焦到清晰仪表盘，并预留一句 CTA 标题空间。 business-ready image for a growth marketer. Goal: Create a high-converting hero image for a search-ad landing page. Deliverable: Search ad landing hero. Visual direction: A clean comparison scene showing the old manual workflow fading into a focused dashboard, with one short CTA headline space. Use precise layout language, keep text short and legible, show realistic work context, include enough negative space for website cropping, avoid generic stock-photo smiles, fantasy elements, celebrity likenesses, copyrighted brands, unreadable text, clutter, warped hands, and exaggerated claims.
