# Product roadmap poster

**Target user:** founder

**Use case:** Create a product roadmap poster for the next two quarters

## Prompt

Create a 路线图视觉，含 now-next-later 泳道、主题标签、依赖关系和客户价值标注。 business-ready image for a founder. Goal: Create a product roadmap poster for the next two quarters. Deliverable: Product roadmap poster. Visual direction: Roadmap visual with now-next-later lanes, theme tags, dependencies and customer-value annotations. Use precise layout language, keep text short and legible, show realistic work context, include enough negative space for website cropping, avoid generic stock-photo smiles, fantasy elements, celebrity likenesses, copyrighted brands, unreadable text, clutter, warped hands, and exaggerated claims.
