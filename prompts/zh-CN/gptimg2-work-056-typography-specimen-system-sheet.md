# 字体样张与系统规范页

**目标用户：** 设计系统负责人

**使用场景：** 记录产品 UI 字阶、数字样式和可访问性说明

## Prompt

为设计系统负责人生成一张 4:3 的真实工作场景图片。目标：记录产品 UI 字阶、数字样式和可访问性说明。交付物：字体样张与系统规范页。视觉方向：A design-system documentation spread with type scale, numeric table samples, button labels, chart annotation examples, contrast notes, spacing tokens and baseline grid hints.。版式与内容要求：use placeholder text and neutral labels only；include headline, body, numeric and UI component examples；show accessibility and contrast notes without legal fine print；use a precise documentation grid；keep typography crisp and readable。请使用明确的版式语言、专业工作语境、清晰可读的短标签、必要的界面或文档细节，并保留适合网站卡片和 GitHub 预览裁切的留白。避免：decorative poster excess, fake logos, tiny unreadable samples, inconsistent baselines, celebrity references, clutter。
