**使用场景：** 为平台商品创建一致的详情图库视觉系统

## Prompt

为电商运营生成一张16:9的真实工作场景图片。目标：为平台商品创建一致的详情图库视觉系统。交付物：电商平台商品图套组。视觉方向：An eight-card listing gallery system covering hero, features, dimensions, use case, comparison, included items, material detail and care instructions.。请使用明确的视觉设计语言：说明视觉焦点、网格、层级、字体、色彩角色、间距、标注、裁切安全区和真实工作语境。文字要短且清晰可读，让这张图真正可用于专业评审、营销活动、交付说明或经营会议。避免真实品牌名、复制 logo、名人肖像、隐私数据、虚假承诺、乱码小字、杂乱构图、畸形 UI、图库式人物和纯娱乐装饰风格。
