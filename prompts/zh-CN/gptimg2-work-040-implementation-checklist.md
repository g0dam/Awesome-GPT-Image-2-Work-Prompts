# 实施清单

**目标用户：** 客户成功经理

**使用场景：** 企业客户上线实施清单视觉

## Prompt

为客户成功经理生成一张 4:5 的工作场景图片。目标：企业客户上线实施清单视觉。交付物：实施清单。视觉方向：清单视觉，含阶段、负责人、依赖、完成状态和可靠项目管理语气。。请使用明确的版式语言，文字短且清晰可读，呈现真实工作语境，保留足够留白方便网站裁切；避免通用图库式假笑、奇幻元素、名人肖像、受版权保护品牌、乱码文字、杂乱构图、畸形手部和夸大承诺。
