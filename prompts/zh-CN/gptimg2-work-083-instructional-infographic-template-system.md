# 教学信息图模板系统

**目标用户：** 学习设计师

**使用场景：** 为内部培训内容构建可复用信息图模块

## Prompt

为学习设计师生成一张4:3的真实工作场景图片。目标：为内部培训内容构建可复用信息图模块。交付物：教学信息图模板系统。视觉方向：A template kit preview with modules for process steps, comparison, decision tree, checklist, common mistakes, grid rules and accessibility notes.。请使用明确的视觉设计语言：说明视觉焦点、网格、层级、字体、色彩角色、间距、标注、裁切安全区和真实工作语境。文字要短且清晰可读，让这张图真正可用于专业评审、营销活动、交付说明或经营会议。避免真实品牌名、复制 logo、名人肖像、隐私数据、虚假承诺、乱码小字、杂乱构图、畸形 UI、图库式人物和纯娱乐装饰风格。
