# 活动平面图与导视套件

**目标用户：** 社群运营

**使用场景：** 规划专业 Meetup 的场地布局、导视系统和工作人员提示

## Prompt

为社群运营生成一张 16:9 的真实工作场景图片。目标：规划专业 Meetup 的场地布局、导视系统和工作人员提示。交付物：活动平面图与导视套件。视觉方向：A simplified venue map with registration, main stage, breakout tables, demo booths, networking zone, quiet area, sponsor wall, signage samples, badge system and accessibility notes.。版式与内容要求：use neutral event and venue placeholders only；show wayfinding signage and badge color system；include staff cue cards and accessibility notes；keep floor plan labels legible；make it practical for operations planning。请使用明确的版式语言、专业工作语境、清晰可读的短标签、必要的界面或文档细节，并保留适合网站卡片和 GitHub 预览裁切的留白。避免：real venue names, brand logos, crowded party scenes, tiny labels, decorative-only event graphics, confusing floor plans。
