# 文化价值观卡片组

**目标用户：** 组织发展负责人

**使用场景：** 内部门户文化价值观卡片组

## Prompt

为组织发展负责人生成一张 1:1 的工作场景图片。目标：内部门户文化价值观卡片组。交付物：文化价值观卡片组。视觉方向：四张价值观卡片，含职场场景、简短行为示例和一致视觉系统。。请使用明确的版式语言，文字短且清晰可读，呈现真实工作语境，保留足够留白方便网站裁切；避免通用图库式假笑、奇幻元素、名人肖像、受版权保护品牌、乱码文字、杂乱构图、畸形手部和夸大承诺。
