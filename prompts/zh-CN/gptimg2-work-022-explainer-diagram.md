# 解释型图表

**目标用户：** 培训负责人

**使用场景：** 解释 RAG 工作原理的图解

## Prompt

为培训负责人生成一张 16:9 的工作场景图片。目标：解释 RAG 工作原理的图解。交付物：解释型图表。视觉方向：清晰三阶段图，含文档、检索、模型回答和护栏说明，面向非技术员工。。请使用明确的版式语言，文字短且清晰可读，呈现真实工作语境，保留足够留白方便网站裁切；避免通用图库式假笑、奇幻元素、名人肖像、受版权保护品牌、乱码文字、杂乱构图、畸形手部和夸大承诺。
