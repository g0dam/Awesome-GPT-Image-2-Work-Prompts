# 图标风格方向表

**目标用户：** 品牌设计师

**使用场景：** SaaS 分析产品图标风格方向表

## Prompt

为品牌设计师生成一张 3:2 的工作场景图片。目标：SaaS 分析产品图标风格方向表。交付物：图标风格方向表。视觉方向：多行一致图标，含描边规则、填充变体、隐喻说明和真实 UI 表面示例。。请使用明确的版式语言，文字短且清晰可读，呈现真实工作语境，保留足够留白方便网站裁切；避免通用图库式假笑、奇幻元素、名人肖像、受版权保护品牌、乱码文字、杂乱构图、畸形手部和夸大承诺。
