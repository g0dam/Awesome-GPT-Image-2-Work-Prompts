# 董事会更新叙事页

**目标用户：** 创始人

**使用场景：** 创建展示进展、约束和待决事项的月度投资人更新页

## Prompt

为创始人生成一张 16:9 的真实工作场景图片。目标：创建展示进展、约束和待决事项的月度投资人更新页。交付物：董事会更新叙事页。视觉方向：A mature board-deck slide with progress since last update, current constraints, decisions needed, metric cards, milestone timeline, hiring/resource ask and risk register.。版式与内容要求：separate progress, constraints and decisions needed；include metrics, timeline, resource ask and risk register；use fictional placeholders and measured language；make board input requested visually explicit；keep text concise and executive-readable。请使用明确的版式语言、专业工作语境、清晰可读的短标签、必要的界面或文档细节，并保留适合网站卡片和 GitHub 预览裁切的留白。避免：startup hype clichés, real company names, dense paragraphs, dramatic imagery, tiny tables, cluttered board slides。
