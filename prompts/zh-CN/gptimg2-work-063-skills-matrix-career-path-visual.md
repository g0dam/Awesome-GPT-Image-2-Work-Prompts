# 能力矩阵与职业路径图

**目标用户：** 组织发展负责人

**使用场景：** 为岗位族和成长路径创建内部人才规划视觉

## Prompt

为组织发展负责人生成一张 4:3 的真实工作场景图片。目标：为岗位族和成长路径创建内部人才规划视觉。交付物：能力矩阵与职业路径图。视觉方向：A role-family grid with proficiency levels, skill clusters, learning moves, promotion readiness signals and manager coaching prompts.。版式与内容要求：show role families and proficiency levels without ranking employees；include learning moves and coaching prompts；use abstract inclusive avatars rather than real faces；keep tables readable and calm；make it feel like an internal planning tool。请使用明确的版式语言、专业工作语境、清晰可读的短标签、必要的界面或文档细节，并保留适合网站卡片和 GitHub 预览裁切的留白。避免：surveillance vibes, real employee data, ranking shame, cartoon entertainment style, tiny unreadable tables, clutter。
