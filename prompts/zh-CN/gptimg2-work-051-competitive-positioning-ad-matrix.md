# 竞品定位广告矩阵

**目标用户：** 增长营销人员

**使用场景：** 为 B2B SaaS 落地页或投放广告创建竞品定位视觉

## Prompt

为增长营销人员生成一张 16:9 的真实工作场景图片。目标：为 B2B SaaS 落地页或投放广告创建竞品定位视觉。交付物：竞品定位广告矩阵。视觉方向：A two-axis competitive positioning matrix with four quadrants, a highlighted product zone, competitor-neutral placeholders, proof chips, SaaS dashboard fragments and a CTA strip.。版式与内容要求：show a clear two-axis matrix with concise quadrant labels；use competitor-neutral placeholders rather than real brand names；include one highlighted value zone, three proof chips and a CTA area；keep hierarchy readable for both GitHub preview and website card crops；use realistic SaaS UI fragments without copying any product interface。请使用明确的版式语言、专业工作语境、清晰可读的短标签、必要的界面或文档细节，并保留适合网站卡片和 GitHub 预览裁切的留白。避免：real company logos, competitor names, celebrity likenesses, generic stock-photo people, unreadable text, inflated performance claims, cluttered axes, distorted UI。
