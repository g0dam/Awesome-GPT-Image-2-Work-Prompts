# 组件插画风格指南

**目标用户：** 设计系统负责人

**使用场景：** 定义产品空状态和帮助内容中的插画原则

## Prompt

为设计系统负责人生成一张16:9的真实工作场景图片。目标：定义产品空状态和帮助内容中的插画原则。交付物：组件插画风格指南。视觉方向：A component illustration documentation sheet with line weight, radius, perspective, shadow depth, object metaphors, do/dont examples and sizing rules.。请使用明确的视觉设计语言：说明视觉焦点、网格、层级、字体、色彩角色、间距、标注、裁切安全区和真实工作语境。文字要短且清晰可读，让这张图真正可用于专业评审、营销活动、交付说明或经营会议。避免真实品牌名、复制 logo、名人肖像、隐私数据、虚假承诺、乱码小字、杂乱构图、畸形 UI、图库式人物和纯娱乐装饰风格。
