# 数据处理合规情境海报

**目标用户：** 企业培训负责人

**使用场景：** 通过真实工作情境解释数据处理规则

## Prompt

为企业培训负责人生成一张 4:3 的真实工作场景图片。目标：通过真实工作情境解释数据处理规则。交付物：数据处理合规情境海报。视觉方向：A corporate learning poster with three workplace scenarios, each showing risk cue, approved action, escalation path and do-not-do note.。版式与内容要求：cover customer data export, vendor file sharing and AI tool input review；use accessible scenario cards with approval and warning states；keep language practical and non-alarmist；include a short footer checklist；use no real company names。请使用明确的版式语言、专业工作语境、清晰可读的短标签、必要的界面或文档细节，并保留适合网站卡片和 GitHub 预览裁切的留白。避免：fear-mongering, dense legal fine print, real logos, unreadable text, celebrity likenesses, cluttered compliance charts。
