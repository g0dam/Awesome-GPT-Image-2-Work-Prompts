# Contributing

Contributions are welcome if they improve the prompt library without weakening source quality.

Good contributions include:

- new practical work-oriented GPT Image 2 prompts;
- better bilingual wording for existing prompts;
- cleaner JSON fields for website integration;
- corrected category metadata;
- higher-quality generated example images.

Please avoid submitting copied third-party prompts verbatim. If a prompt is inspired by a public example, rewrite it into an original, structured workplace use case.

## Prompt Requirements

Each prompt should include:

- a clear target user;
- a practical work use case;
- a concrete visual deliverable;
- layout and composition guidance;
- text readability constraints;
- negative constraints for quality and safety;
- English and Simplified Chinese versions.

## File Checklist

When adding a prompt, update:

- `data/prompts.json`
- `prompts/en/`
- `prompts/zh-CN/`
- `images/`
- the matching category file under `categories/`
