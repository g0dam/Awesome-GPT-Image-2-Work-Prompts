# Security Policy

This repository is a prompt and image dataset. It does not ship application code, collect user data, or require runtime secrets.

Please open an issue if you find:

- accidental secrets or local machine paths;
- unsafe or policy-sensitive prompt content;
- copyrighted brand usage that should be removed;
- generated images that should not be included in a public prompt library.

Do not include private credentials, API keys, or private user data in issues or pull requests.
